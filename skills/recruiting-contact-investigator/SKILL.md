# Recruiting Contact Investigator

Find the requested number of strong contacts; default to two. Ask only when no company, person, vacancy, or usable source can be inferred.

## Investigate

Read and exhaust every applicable branch in [contact-search-playbook.md](references/contact-search-playbook.md). Run a relentless investigation:

- Extract all clues from the user's evidence.
- Search every plausible source, query, language variant, tool, organizational rung, person, and identity pivot available. Let evidence choose the path; invent new pivots and alternate public routes as leads emerge or fail.
- Pursue anyone with credible hiring influence or referral access. Preserve sources and conflicts.

A contact counts only when independent evidence supports both:

1. **Current role:** the person currently holds a relevant position at the company.
2. **Direct route:** the account or address belongs to that person.

Keep guessed handles, stale roles, snippets, directories, and generic company channels as unresolved leads. Use public information and authorized access.

Completion requires the target plus every obvious stronger lead checked. If the target remains unmet, exhaust every plausible branch and record each dead end.

## Return

Rank verified contacts by hiring influence, relevance, freshness, route confidence, and response likelihood. For each, give the role, direct route, evidence for both checks, and why they rank there. Then list unresolved leads, sources tried, dead ends, generic fallbacks, and the recommended outreach order. Explain any shortfall instead of filling it with guesses.
